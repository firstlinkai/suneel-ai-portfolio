# Suneel | AI Content Architect Portfolio

This is a high-end, minimalist personal portfolio built with React 19, Vite, and Tailwind CSS v4.

## Features
- **Light Minimalist Aesthetic**: Premium design focusing on typography and spacing.
- **Intelligence Layer**: Showcases automated Google Ads Audits and Social Media intelligence.
- **Savings & Efficiency Calculator**: Interactive widget demonstrating AI-driven ROI and time savings.
- **Fully Responsive**: Optimized for all devices.

## Tech Stack
- **Framework**: React 19
- **Bundler**: Vite
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/firstlinkai/suneel-ai-portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Deployment
This repository is ready for deployment on **Vercel**. Simply connect your GitHub account and import this project.
